# Architecture

For TypeScript web, frontend, backend, fullstack contracts, and test strategy decisions, use the `web-app-design` skill and its delegated skills.

Project-local conventions override global architecture preferences. Preserve the project's framework, package manager, test runner, routing, styling, folder structure, and naming style unless the user explicitly asks for migration.

Do not duplicate architecture rules in `AGENTS.md` when a skill already covers them.
